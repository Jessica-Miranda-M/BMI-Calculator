## üõ†Ô∏è Technical Challenges & Evolution

During this project, I focused on identifying technical hurdles and applying industry-standard solutions:

### 1. Dynamic UI & User Experience (UX)
* **Challenge:** Creating a smooth way for users to input height without excessive typing.
* **Solution:** Implemented a **SeekBar** to allow real-time height selection, synchronizing UI components instantly.

### 2. Error Handling & App Stability
* **Challenge:** Preventing app crashes caused by empty fields or invalid numeric inputs.
* **Solution:** Applied **Try-Catch** blocks and string validation to ensure the app remains stable and provides user-friendly feedback via Toasts.

### 3. Resource Management (Clean Code)
* **Challenge:** Avoiding hardcoded values that make maintenance difficult.
* **Solution:** Centralized all strings and colors in **XML resource files** (`strings.xml`, `colors.xml`), following Android best practices for scalability.

### 4. Advanced Version Control
* **Challenge:** Solving synchronization conflicts between local and remote repositories.
* **Solution:** Mastered commands like `git pull --rebase` to maintain a clean, linear, and professional project history.
